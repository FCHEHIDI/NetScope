# ADR — Architecture Decision Records

Ce dossier contient les décisions d’architecture structurantes pour NetScope.

## Convention

- Format: `ADR-XXXX-titre-court.md`
- Statuts: `proposed`, `accepted`, `superseded`
- Une ADR par décision majeure
- Toute modification d’invariant ou de contrat doit référencer une ADR
